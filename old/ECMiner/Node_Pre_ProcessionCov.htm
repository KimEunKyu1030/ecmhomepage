행렬변환

행과 열을 교환하여 입력 데이터 테이블을 변환합니다. 새 행 이름은 기본으로 설정되거나 이전 특정 행의 데이터로 설정할 수 있습니다. 새 열의 타입은 기본적으로 가장 명확한 타입이 되며, 한 행의 모든 셀에 적용됩니다.

사용법, 속성

사용법

  • 행렬변환 노드는 다음과 같은 속성들을 설정할 수 있습니다.

  • 이전 노드의 데이터 테이블의 필드명을 데이터로 변환하려면 var_name열 생성여부 속성을 지정합니다.
  • 특정 열의 데이터를 변수명으로 변화하려면 변수명 지정여부 속성을 지정합니다.
  • 변수명 지정여부 속성을 지정하면 변수명으로 사용할 이전 노드 데이터 테이블의 필드명을 선택할 수 있습니다.
  • 동일한 병수명이 있는 경우에 대한 처리를 변수명 중복 시 속성을 통해 설정할 수 있습니다.
  • 이전 노드로부터 변수명을 제대로 가져오지 못한 경우 [변수정보 가져오기] 버튼을 눌러 다시 가져올 수 있습니다.


    • 다음과 같이 판매지역이 위쪽에 나열되어 있고 분기가 왼쪽에 나열되어 있는 데이터를 예를 들면


    열과 행을 회전하여 다음과 같이 분기를 위쪽에 나열하고 지역을 옆쪽에 나열할 수 있습니다.
