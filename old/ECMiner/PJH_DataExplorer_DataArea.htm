데이터영역 (데이터 탐색기)

데이터가 보여지는 영역으로, 데이터 탐색 및 행, 열 편집이 가능합니다.

윈도우 배치

Figure 데이터 영역

윈도우 설명

윈도우 번호 윈도우명 기능 비고
1 열 머리글 데이터 영역의 위쪽 부분으로, 열 번호, 필드명이 보여집니다. 마우스 오른쪽 메뉴를 이용하여 열 숨기기,열 삭제, 복사, 삭제, 전체선택 등이 가능합니다. 열의 경계 부분을 더블 클릭하면 열 간격이 자동으로 맞춰집니다.
2 행 머리글 데이터 영역의 왼쪽 부분으로, 행 번호가 보여집니다. 마우스 오른쪽 메뉴를 이용하여 행 삭제, 복사, 삭제, 전체선택 등이 가능합니다.
3 데이터 영역 실제로 데이터가 표시되는 부분입니다. 처음에는 데이터시트만 존재하나, 데이터 분석에 따라, 데이터 분포, 이산형 분포, 상관 관계등의 시트가 추가 됩니다. 마우스 오른쪽 메뉴를 이용하여 복사, 삭제, 전체선택 등이 가능합니다.(데이터 영역의 종류에 따라 메뉴는 조금씩 달라집니다.)
4 시트 탭 데이터 영역을 관리하는 탭입니다. 선택을 통해 원하는 데이터 영역을 볼 수 있을 뿐만 아니라, 마우스 드래그를 사용하여 탭의 위치를 바꿀 수 있습니다.
