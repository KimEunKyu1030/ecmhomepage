연결함수

로지스틱 회귀분석은 범주형 설명변수에 대해 반응변수의 관계를 유연한 분석하는 방법론입니다. 로짓스틱 분석은 설명변수에 대한 확률을 결정하는 함수로 로짓을 사용한 방법론 입니다. 일반적으로 이항로지스틱 분석에는 로짓이외 다른 여러가지 함수를 사용할 수 있습니다.

ECMiner™에는 로짓 이외에 노밋과, 곰핏에 대한 연결함수 기능을 지원합니다

    참조 - 교호작용 추가 방법

    교호작용은 파생변수를 이용하여 추가할 수 있습니다.


    <범주형 대 범주형의 교호작용 >

    파생변수를 추가하여 교호작용 효과를 낼 수 있습니다. 예를 들어 FIELD1은 {"A", "B", "C"}, FILED2은 {"a", "b"}로 구성된 필드의 교호작용을 추가한다고 가정하면, 아래와 같이 파생변수에 수식편집기를 입력하여 교호작용을 추가할 수 있습니다.


    IF {FIELD1} = "A" and {FIELD2} = "a" THEN
    "Aa"
    ELSEIF {FIELD1} = "A" and {FIELD2} = "b" THEN
    "Ab"
    ELSEIF {FIELD1} = "B" and {FIELD2} = "a" THEN
    "Ba"
    ELSEIF {FIELD1} = "B" and {FIELD2} = "b" THEN
    "Bb"
    ELSEIF {FIELD1} = "C" and {FIELD2} = "a" THEN
    "Ca"
    ELSEIF {FIELD1} = "C" and {FIELD2} = "b" THEN
    "Cb"
    ENDIF

    <범주형 대 수치형의 교호작용>

    각 범주에 대해 파생변수를 추가하여 교호작용 효과를 낼 수 있습니다. 예를 들어 FIELD1은 {"A", "B", "C"}, FILED2은 수치형으로 구성된 필드의 교호작용을 추가한다고 가정하면, 아래와 같이 파생변수에 수식편집기를 입력하여 교호작용을 추가할 수 있습니다.


    # FIELD1중 - A에 대한 처리
    IF {FIELD1} = "A" THEN
    {FIELD2}
    ELSE
    0
    ENDIF

    # FIELD1중 - B에 대한 처리
    IF {FIELD1} = "B" THEN
    {FIELD2}
    ELSE
    0
    ENDIF

    # FIELD1중 - C에 대한 처리
    IF {FIELD1} = "C" THEN
    {FIELD2}
    ELSE
    0
    ENDIF

    < 수치형 대 수치형의 교호작용 >

    파생변수를 추가하여 교호작용 효과를 낼 수 있습니다. 예를 들어 FIELD1과, FILED2가 수치형으로 구성된 필드의 교호작용을 추가한다고 가정하면, 아래와 같이 파생변수에 수식편집기를 입력하여 교호작용을 추가할 수 있습니다.


    {FIELD1}*{FIELD2}
