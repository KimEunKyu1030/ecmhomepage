비중심 카이 제곱 분포

iid 일 때, 는 자유도가 k인 카이제곱 분포를 따르는 확률 변수입니다.
하지만 다음과 같이 변형을 하면 비중심 카이제곱 분포를 따른다고 합니다.
일반적으로 비중심 카이제곱 분포에서 사용되는 parameter는 입니다.

  • 비중심 카이 제곱 분포의 pdf

    • 는 Bessel function


  • 비중심 카이 제곱 분포의 cdf

    • 는 incomplete gamma function


  • 비중심 카이 제곱 분포의 mgf

    • Mgf(moment generating function)은 Moment를 구하는데 매우 유용한 함수입니다. 비중심 카이 제곱 분포의 mgf는 다음과 같습니다.


  • 비중심 카이 제곱 분포의 평균,분산


  • 비중심 카이 제곱 분포의 누적 분포 함수의 역함수


  • 예시
    • 일 때 pdf,cdf 그래프


    • 일 때 평균, 분산. 일 때 pdf, cdf


    • 일 때 누적 분포 함수의 역함수
