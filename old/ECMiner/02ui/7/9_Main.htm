일반화 극단값 분포

일반화 극단값 분포는 연속 확률 분포의 일종입니다. 이 분포는 Gumbel, Weibull, Frechet 분포와 관련이 있습니다. 분포에 쓰이는 즉 k의 값에 따라 k=0일 때 Gumbel, k>0일 때 Frechet, k<0일 때 Weibull분포라고 합니다. 이를 각각 type 1, type 2, type 3 극단값 분포라고도 부릅니다.
