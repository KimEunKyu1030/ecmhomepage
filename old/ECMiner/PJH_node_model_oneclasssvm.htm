One Class SVM 모델 노드

One class SVM모델 노드는 One class SVM 분석 결과로 생성됩니다. Training 데이터 셋으로 이상치를 탐지하는 Support vector를 찾고, 이를 이용하여 새로운 데이터 셋에 대해 이상치를 판별할 수 있습니다.

  • 옵션 정보

    • One class SVM를 이용해서 모델링 시 어떤 옵션을 사용했는지를 보여줍니다. 이것을 통해 옵션이 바뀌었을 때 결과가 어떻게 바뀌었는지 판단할 수 있습니다


  • Support Vector List

    • 모델링 결과 선정된 support vector의 개수와 리스트를 보여줍니다. 이는 모델링 수행 결과 alpha값이 0보다 큰, 즉 support vector로 선정된 데이터 값과 alpha값을 보여줍니다.
