LOGISTIC 모델 노드

LOGISTIC 모델 노드는 트레이닝 데이터를 러닝하여 새로운 데이터의 분류에 사용될 수 있습니다.

  • 클래스수와 총시행수

    • 종속변수의 클래스 수와 알고리즘 실행할 때의총 시행수를 알 수 있습니다.

  • Y Value

    • 종속변수 Y의 클래스별 빈도 및 비율을 보여줍니다.


  • 로지스틱테이블

    • Coef는 각 변수의 계수값을 나타내며, SECoef는 각 계수(Beta)의 standard error입니다. 각 변수 유의성 검정의 z-value, p-값을 보여줍니다. p-값이 유의수준보다 작을때 유의한 변수라고 판단합니다. Odds Ratio는 입력변수가 분류결정에 미치는 영향의 정도를 나타내는 값입니다. 오즈비가 1보다 작다는 것은 입력변수 xi가 (-)의 영향을 준다는 것이고, 오즈비가 1보다 크다는 것은 (+)영향을 줌을 의미합니다.

    값이 1.#IO, 1#QO 또는 1.#INF와 같이 나타난 경우 값이 너무 크거나 작은 경우입니다.

  • Confusion matrix for number & percentage

    • 종속변수의 원래값과 Logistic Regression을 통해 얻어진 예측값 간의 빈도, 퍼센트 교차표를 보여줍니다. Confusion Matrix를 통해 각 범주에 대한 분류정확도를 판단할 수 있습니다.
