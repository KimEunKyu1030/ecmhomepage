변환함수
  • 예시에 표현되는 결과값 형태는 함수 적용 결과 데이터 형태가 문자형일 경우 "결과값"의 형태로, 날짜형일 경우 #결과값#의 형태로 표현됩니다.(실제로 화면상에 표현되는 값은 ",#을 제외한 숫자/문자만 표현됩니다.)
    • 함수 인수 설명 예
    ASC(text) text: 문자열 문자열(text) 중 첫 문자의 ASCII 코드값을 반환합니다. ASC("ABC") -> 65
    CHR(num) num: ASCII 코드 입력된 ASCII 코드에 해당하는 문자를 반환합니다. CHR(65) -> "A"
    DATETOSTR(date, fmt) date: 날짜형, fmt: 포맷 날짜시간형을 'yyyy-mm-dd HH:MM:SS' 과 같이 지정한 포맷 에 따라 문자형 형식으로 변경합니다. 년도는 y, 월은 m, 일은 d, 시간은 H, 분은 M, 초는 S로 대변됩니다. 예를 들어 문자열 '2004-10-23 23:28:11'의 경우 'yyyy-mm-dd HH:MM:SS'와 같이 fmt를 지정합니다. fmt가 없을 경우 'yyyymmdd'로 인식합니다. DATETOSTR(#2004-01-01 12:10:10#, 'yyyymmdd-HHMMSS) -> "20040101-121010"
    IIF(cond, expr1, expr2) cond: 조건문
    expr1: 수식1
    expr2: 수식2     		주어진 조건문(cond)이 참이면 수식1(expr1), 거짓이면 수식2(expr2)를 계산합니다. IIF(1 > 0, 1 + 2, 1 + 3) -> 3
    INT(num) num: 실수 입력된 실수를 정수로 변환합니다. 소수점 이하의 숫자가 모두 무시됩니다. INT(2.371) -> 2
    STR(num [, num1 [,num2]]) num: 숫자
    num1: 최대 길이
    num2: 소수점 자리수     		입력된 숫자(num)을 문자로 변환합니다. num1은 변환할 때의 최대 길이, num2는 소수점 자리수이며 필수 인수는 아닙니다. STR(123.345, 6, 2) -> "123.35" (반올림된 결과 나옴)
    STRTODATE(text, fmt) text : 문자열, fmt : 포맷 fmt에 지정한 포맷에 따라 text를 날짜형으로 변환합니다. 년도는 y, 월은 m, 일은 d, 시간은 H, 분은 M, 초는 S로 대변됩니다. 예를 들어 문자열 ‘2004-10-23 23:28:11’의 경우 ‘yyyy-mm-dd HH:MM:SS’와 같이 fmt를 지정합니다. STRTODATE("2004-01-01 11:50",’yyyy-mm-dd HH:MM’) -> #2004-01-01 11:50:00#
    VAL(text) text : 문자열 문자열(text)을 숫자로 변환합니다. VAL("123") -> 123
