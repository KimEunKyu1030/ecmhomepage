모델 데이터 추출

이 기능을 통해 모델 특정적인 각종 데이터를 추출할 수 있습니다. 이렇게 추출된 데이터는 다시 읽혀져 전처리 등으로 재사용 될 수 있습니다.

사용법

  • 저장할 파일명을 입력하거나 선택합니다


  • 저장된 파일을 자동으로 읽어올 것인지 결정합니다.


  • 읽어온 데이터를 확인하거나 가공합니다.


모델별 추출 데이터 유형

    모델명 데이터 유형 기타
    HIERARCHICAL 군집 중심
    KMEANS 군집 중심
    LDA 각 클래스별 평균, 각 클래스별 선형판별함수 계수 C:클래스
    LOGISTIC 로지스틱 회귀테이블 SE: SE Coef
    MLR 회귀테이블
    PCA 독립변수 로딩 PC:주성분
    PCR 독립변수 로딩, 독립변수 회귀계수 PC:주성분
    PLS 잠재변수 로딩, 잠재변수 가중치 LV:잠재변수
    QDA 각 클래스별 평균
    RBF 군집 중심

  • HIERARCHICAL
    각 군집 중심 정보와 전체 관측치 중심 정보를 나타내고 있습니다.


  • KMEANS
    각 군집 중심 정보를 나타내고 있습니다.


  • LDA
    각 변수의 클래스별 판별함수 상수와 계수를 나타냅니다. 그리고 각 변수의 클래스별 평균을 나타내고 있습니다.(C:클래스)


  • LOGISTIC
    로지스틱 테이블 정보를 나타내고 있습니다.


  • MLR
    회귀 테이블 정보를 나타내고 있습니다.


  • PCA
    주성분별 입력변수 로딩값을 나타내고 있습니다.(PC:주성분)


  • PCR
    주성분별 입력변수 로딩값과 계수를 나타내고 있습니다.(PC:주성분)


  • PLS
    잠재변수별 로딩,가중치 값과 각 종속변수에 대한 Coefficient Beta값을 나타내고 있습니다.(LV:잠재변수)


  • QDA
    각 변수의 클래스별 평균을 나타내고 있습니다.


  • RBF
    각 군집 중심 정보를 나타내고 있습니다.
