폴더 실행
특정 폴더에 있는 동일한 형식의 다중 파일을 대상으로 동일 프로젝트를 반복 실행하는 기능입니다. 다음과 같이 실행할 수 있습니다.
(1) 실행할 프로젝트를 작업창에 열어 놓습니다.
(2) "메뉴>스트림>폴더 실행"을 선택합니다.
(3) 다음과 같은 폴더 실행 화면이 나타납니다.


폴더 실행을 다음과 같은 순서로 나누어 설명합니다.
    1. 입력 폴더
    2. 필터 옵션
    3. 입출력 설정
    4. 입력 노드 설정
    5. 출력 노드 설정
    6. 버튼

1. 입력 폴더

  • 입력 폴더의 실행 대상 데이터가 존재하는 폴더를 (1)영역에 직접 입력하거나, (2)번 아이콘을 클릭하여 폴더 브라우저에서 선택합니다. 아래와 같은 폴더 브라우저 화면이 실행됩니다.

2. 필터 옵션

  • 필터 옵션은 체크박스를 선택하여 파일 확장자를 지정하거나, 파일 이름을 지정할 수 있습니다.(""?,*"사용 가능) 폴더 실행은 입력폴더 내의 사용자가 설정한 필터옵션과 일치하게 선택된 파일의 개수 만큼 프로젝트를 반복 실행합니다.

3. 입출력 설정

  • 입출력 설정은 파일입력과 파일출력 노드 이름을 사용하므로 프로젝트 생성 시 노드 이름을 유일하게 부여해야 합니다. 입출력 설정 시, 입력 노드는 1개, 출력 노드는 여러 개 설정할 수 있습니다. 설정된 입력노드의 파일이 입력폴더 내의 사용자가 설정한 필터 옵션과 일치하여 선택된 파일로 대체됩니다.

4. 입력 노드 설정

  • 입력 노드 설정 버튼을 클릭하면 입력 노드 설정 화면이 나타납니다. 입력 노드 이름은 열어 놓은 프로젝트에 존재하는 입력 노드 이름을 목록으로 보여줍니다. 사용자는 표시되는 항목 중 사용할 입력 노드를 선택할 수 있습니다.

5. 출력 노드 설정

  • 출력 노드 설정 버튼을 클릭하면 출력 노드 설정 화면이 나타납니다. 출력노드 이름, 출력 폴더, 파일 이름을 입력합니다. 출력 노드 추가 시, "출력 노드 설정" 버튼을 선택하여 추가 입력할 수 있습니다. 출력 노드 삭제 시, 입출력 노드 목록에서 파일 출력 노드를 선택 후, "출력 노드 삭제" 버튼을 클릭하여 출력 노드를 목록에서 삭제합니다.

6. 버튼

    폴더 실행 설정 값 내보내기, 가져오기 버튼과 폴더실행 옵션 설정 버튼, 폴더 실행 취소 버튼, 설정된 값을 기반으로 폴더 실행을 수행하는 확인 버튼을 제공합니다.

  • 내보내기
    • 폴더 실행을 위해 설정한 입력 폴더, 필터 옵션, 입출력 설정 등의 값을 XML 파일로 저장합니다.
  • 가져오기
    • 폴더 실행 설정 값이 저장된 XML 파일을 가져옵니다.
  • 폴더 실행 옵션
    • 실행 결과 로그 파일 저장 옵션을 설정합니다.



    1. 파일(file)
      2. 로그 파일이 저장될 경로로 초기 설정값에 ECMiner 설치 위치로 설정합니다.
    2. 저장 주기(rotation)
      3. 로그를 저장하는 파일이 생성되는 주기입니다. 일 단위, 주 단위, 월 단위로 설정할 수 있고 초기값은 주기가 없도록(never) 설정됩니다. 예를 들어, 일 단위로 설정할 경우, 해당 일에 실행한 결과 로그는 한 개의 파일에 함께 저장됩니다.
    3. 파일명(archive)
      4. 로그 파일 이름을 설정하는 방법으로 숫자로 설정하는 방법과 시간으로 설정하는 방법이 있습니다. 초기값에 시간으로 파일 이름이 설정됩니다.
    4. 로그 수준(level)
      5. 로그에 저장될 내역의 수준으로 지정된 수준 이상의 로그만 기록합니다. none~trace까지 9개의 레벨이 존재하며 trace로 설정한 경우, 가장 많은 로그 기록이 출력됩니다.
    5. 순환 시간(times)
      6. 로그 파일 순환의 기준이 되는 시간 관련 설정으로 초기값에 로컬 시간으로 설정됩니다.
    6. 로그 저장 포맷(format)
      7. 로그가 저장될 때의 형태로 초기값에 "[년-월-일 시:분:초]로그내역"형태로 저장되도록 설정되었습니다.
  • 취소
    • 폴더 실행을 취소합니다.
  • 확인
    • 입력 폴더, 필터옵션, 입출력 설정 등의 설정된 값을 기반으로 폴더 실행을 수행하고, 진행 상태를 화면에 보여줍니다.
