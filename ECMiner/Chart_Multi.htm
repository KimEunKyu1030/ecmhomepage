멀티 차트

멀티 차트 노드는 여러 변수들의 특징을 한 눈에 보기 쉽도록 고안한 ECMiner 고유의 차트입니다.

사용법, 속성, 기본 화면, 사용 예시

사용법

  • 차트형태는 Trend, Data, Box Plot, Distribution, Correlation 중에 하나를 선택합니다.
    • Trend : 데이터를 시간 순서대로 표현하는 기능.
    • Data : 데이터를 단순하게 산점도로 표현하는 기능.
    • Box Plot : Box Plot을 그려주는 기능.
    • Distribution : 변수의 분포를 정규 분포 확률 플롯으로 표현하는 기능.
    • Correlation : 변수간 관계를 표현하는 기능.
  • 출력 옵션은 화면에 가로(행) 및 세로(열)에 몇개의 표현할 차트를 보여줄지 입력합니다.
  • X축 변수
    • Label : 차트형태가 Data일 때에만 활성화됩니다. 이산형 변수만이 선택 가능하며, 이산형 변수를 X축으로 하여 차트를 그립니다.
    • X축 변수 : 차트형태가 Trend, Correlation일 때에만 활성화됩니다. Trend는 날짜형 변수만을 선택할 수 있으며, Correlation은 연속형 변수만을 선택할 수 있습니다.
  • Y축 변수 속성창에서 사용할 하나 이상의 Y변수를 선택합니다.
  • 선택사항
    • 관리한계 UCL, LCL 변수 선택
    • 차트분할 변수 선택
    • 그룹변수 변수 선택

기본 화면

  • 노드 속성창에서 작성한 속성값들이 멀티차트 화면 옵션들에 입력되어 있습니다.
  • 멀티차트 화면에서 옵션의 수정이 가능합니다. 화면에서 옵션을 수정하면 노드 속성창에도 반영이 됩니다.
