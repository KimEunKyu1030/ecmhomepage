DNN 모델 노드

DNN 모델 노드는 DNN 모델링 결과로 생성됩니다. Training 데이터 셋으로 DNN 분석을 수행한 후 분석 결과인 DNN 모델 노드를 Test 데이터 셋(새로운 데이터 셋)에 적용하여 새로운 데이터의 예측/분류에 사용될 수 있습니다.

  • 옵션정보

    • 모델링 방법(예측/분류), 최적화기법 정보, 시행수 등 인공신경망 학습 전반에 걸친 학습 정보를 저장합니다.


  • 계층 정보

    • 인공신경망의 입력층, 각각의 은닉층, 출력층에서 노드(퍼셉트론) 정보를 보여줍니다.


  • 가중치 정보
    • 입력층 -> 첫번째 은닉층

    입력층의 모든 노드들과 첫번째 은닉층의 모든 노드들의 연결에 대한 가중치를 보여줍니다. 행은 입력층의 노드, 열은 첫번째 은닉층의 노드들을 의미합니다.


    은닉층 n -> 은닉층 n+1

    은닉층 n의 모든 노드들과 은닉층 n+1의 모든 노드들의 연결에 대한 가중치를 보여줍니다. 행은 은닉층 n의 노드, 열은 은닉층 n+1의 노드들을 의미합니다.


    마지막 은닉층 -> 출력층

    마지막 은닉층의 모든 노드들과 출력층의 모든 노드들의 연결에 대한 가중치를 보여줍니다. 행은 마지막 은닉층의 노드, 열은 출력층의 노드들을 의미합니다.


  • ANOVA 테이블

    • 회귀분석의 경우에만 해당됩니다. 독립변수가 종속변수 y를 설명하는데 있어서 통계적으로 유의한지 검정하기 위해 F검정을 합니다. p-값이 유의수준보다 커서 유의하지 않다면 회귀계수에 대한 해석은 의미가 없습니다.


  • 표준에러정보

    • 회귀분석의 경우에만 해당됩니다.

    1. R-square(결정계수)
      도출된 회귀식이 측정값들을 대표할만한가를 확인하는 것으로, 즉, 추정된 회귀선이 관측치들을 얼마나 잘 설명하는지를 검정하는 값입니다. 1에 가까울수록 회귀식이 데이터를 잘 설명하고 있음을 의미합니다.
    2. Root-MSE (Root-Mean Square Error)

    3. MAE (Mean Absolute Error)

    4. MPE (Mean Absolute Percentage Error)

    2~4 번 모두 종속변수의 원래값과 추정값의 오차를 나타낸 것으로 값이 작을수록 좋은 모형이라할 수 있습니다.
