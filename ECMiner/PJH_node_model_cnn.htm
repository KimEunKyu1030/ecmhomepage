CNN 모델 노드

CNN 모델 노드는 CNN 모델링 결과로 생성됩니다. Training 이미지 데이터 셋으로 CNN 학습을 수행한 후 분석 결과인 CNN 모델 노드를 Test 이미지 데이터 셋(새로운 데이터 셋)에 적용하여 새로운 이미지 데이터의 분류에 사용할 수 있습니다.

  • 옵션정보

    • 모델링 네트워크 종류, 패딩 설정, 최적화기법 정보, 시행수 등 해당 합성곱 신경망 모델 학습에 사용한 주요 설정들을 보여줍니다.


  • 오분류 정보

    • 모델링 완료 후 얻어진 최종 모델 구조를 모델링용 데이터에 적용하여 얻어진 분류 결과 및 정확도입니다. 표 하단에는 전체 오분류 개수 및 오분류율이 표시됩니다.
