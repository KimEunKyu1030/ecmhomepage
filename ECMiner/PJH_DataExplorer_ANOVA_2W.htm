이원 배치법: Stack

이원 배치 분산분석은 두 가지 인자에 대한 분산 분석입니다. 인자1과 인자2, 관측치를 선택하면 분석을 하실 수 있습니다.

분산 분석 다이얼로그 열기

  • 메뉴의 [분석]을 선택한 후, [분산분석]을 다시 선택하면 3가지 분산 분석 종류가 나열됩니다. 그중에서 이원배치법:Stack을 선택하시면 됩니다.

Stack 데이터

  • Stack 데이터 : 아래의 Figure 1과 같이 데이터를 하나의 필드에 쌓아서 정리한 데이터 형태를 말합니다. Figure 1를 보면 두번째 인자인 온도가 세 가지 값을 가지지만, 이들을 하나의 필드에 쌓아서 표시된 형태를 볼 수 있습니다.

이원 배치 : Stack 데이터

  • 인자 1 : "밝기" 필드 --> 40, 60, 80 lux의 밝기 변화를 주었다.
  • 인자 2 : "온도" 필드 --> 15, 20, 25의 온도 변화를 주었다.
  • 관측치 : "결과" 필드 --> 세번째 결과 필드가 관측치이다.
    • Figure 1 이원 배치 : Stacked 데이터

인자 및 관측치 선택방법

  • 마우스 드래그를 이용하여 선택할 수 있습니다. 먼저 왼쪽의 필드 리스트에서 필드를 선택한 후, 오른쪽의 인자 리스트 또는 관측치 리스트로 마우스로 드래그앤드롭을 하면 필드가 왼쪽에서 오른쪽 리스트로 이동됩니다.
