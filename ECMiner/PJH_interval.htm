구간화 노드

구간화 노드는 종속변수에 미치는 영향도에 따라 독립변수를 여러 개의 구간으로 나누는 노드입니다.

사용법, 속성

사용법

  • 연속형 변수를 범주형 변수로 변환할 때 사용하며 결측치 처리에 용이합니다.
  • 형태변경 노드에서 종속변수를 선택합니다.
  • 여러 개의 독립변수를 선택합니다.

    • Event : 2가지 종속변수의 값 중에서 어떤 것을 Event로 인식할 지를 선택합니다.
    • Independent Var: 여러 개의 독립변수에 대해 각각 구간화를 실행합니다.
    • Interval: 구간의 간격이 동일한 Bucket과 구간의 수가 동일한 Quantile 방식 중 선택합니다.
    • Interval Number: 구간 수를 설정합니다.
  • Run 버튼을 누르면 아래와 같은 차트 및 통계량이 나타납니다.
    • NOTE
    Interval과 Interval Number는 연속형 변수에만 해당합니다.
    • 연속형 변수일 경우 화면

  • 이산형 변수는 Interval, Interval Number 설정 없이 바로 Run 버튼을 누르면 모든 속성을 각각의 그룹으로 생성합니다.
    • 이산형 변수일 경우 화면

  • Sum: 2개 이상의 구간을 병합합니다.
    • 연속형 데이터는 인접 구간끼리만 병합할 수 있습니다.
    • 이산형 데이터는 인접 구간이 아니더라도 병합할 수 있으며조건은 or로 변경됩니다.
      (예: A2=a와 A2=b를 병합하면 A2=aor b)
  • Div: 1개의 구간을 2개 이상으로 분할합니다.
    • 연속형 데이터는 구간 내의 값을 입력해야 합니다. (2구간일 경우 21~28 사이의 수 입력)
    • 이산형 데이터는 모든 속성을 각각의 그룹으로 분할합니다.
