기본 데이터베이스 설정
도구(T) - 옵션(O)의 기본 데이터베이스 설정을 선택하면 다음과 같은 옵션창이 뜨게 됩니다. ECMiner™가 지원하는 데이터베이스는 ODBC와 OLE DB입니다. 아래 옵션 창에서 드라이버 및 로그인 정보를 입력하면 프로젝트창에서 데이터베이스 작업 시 기본 데이터베이스 정보로 설정되어 편리하게 이용할 수 있습니다.

번호 설명
① OLE DB 관련 노드에 입력할 데이터베이스 접속 정보를 입력합니다.
② ODBC 관련 노드에 입력할 데이터베이스 접속 정보를 입력합니다.
③ 이 항목을 선택해 놓아야 데이터베이스 관련 노드를 추가할 때 데이터베이스 접속 정보가 자동 입력됩니다.

관련 항목

  • ODBC 입력 노드
  • OLEDB 입력 노드
  • ODBC 출력 노드
  • OLEDB 출력 노드
