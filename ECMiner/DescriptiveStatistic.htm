기술통계

데이터 탐색기에서는 기술 통계기능을 통해, 선택된 변수에 대한 기술 통계량을 볼 수 있습니다.

실행 방법

  • [분석] - [기술통계]를 선택하거나, 또는 툴바의 기술통계 아이콘을 클릭하시면 기술통계 다이얼로그가 나타납니다.

기술통계 다이얼로그

  • 1 : 여러 변수의 기술 통계량을 볼 때 사용합니다. 추가 버튼을 누르면 변수를 추가할 수 있는 리스트가 추가되고, 삭제 버튼을 누르면 마지막 변수 리스트가 제거됩니다. 만약 변수 리스트 원소가 1개일 때는 삭제 버튼을 눌러도 더 이상 리스트 원소가 줄어들지 않습니다.
  • 2 : 기술 통계량을 볼 변수를 선택할 수 있는 콤보 박스입니다.
  • 3 : 변수 선택 버튼. 변수를 선택하는 다른 방법으로, 버튼을 누른 후 데이터 영역의 열 머리글을 클릭하면 변수가 선택됩니다.
  • 4 : 전체 선택 버튼. 통계량 선택 시, 체크박스로 일일이 선택하지 않고, 모든 통계량을 한번에 선택 또는 해제 할 수 있는 버튼 입니다.
NOTE 1
수학적인 통계량을 얻을 수 없는 범주형 데이터변수는 선택할 수 없습니다.(변수 선택 콤보 박스에 범주형 변수는 나타나지 않습니다)

실행 결과

결과 설명

선택된 변수들에 대해 통계량을 보여줍니다. 지원되는 통계량으로는 총합, 평균, 최소값, 최대값, 범위, 분산, 표준편차, 왜도, 첨도, 중앙값, 4분위수가 있습니다.
