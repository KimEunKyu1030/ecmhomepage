이미지 통계량

  • 이미지 통계량노드는 이미지를 구성하는 pixel들의 통계량 정보를 추출하는 기능을 수행합니다.

    • 개요, 고려사항, 사용법, 속성, 결과

    개요

    • 이미지 통계량 노드는 디지털 이미지에 대하여 자체적으로 회색조(grayscale) 전처리를 수행합니다.
    • 전처리 후 얻어진 회색조 이미지에 대하여 이미지를 구성하는 pixels 각각이 가지는 grayscale values(0 이상 255 이하의 정수)에 대한 통계량(평균, 최대, 최소, 중간, 분산, 표준편차)을 추출합니다.
    • 출력 노드 또는 차트 노드를 사용하여 추출된 통계량을 시각화할 수 있습니다.

    고려사항

    • 노드 속성창의 이미지 입력 - 입력 이미지 경로에는 유효한 이미지 파일 경로 field를 지정해야 합니다.

    사용법

    • 이미지 입력 노드를 통해 이미지 데이터 파일들을 읽어들입니다.
    • 원본 이미지에 대하여 통계량을 추출하고자 한다면 이미지 입력 노드에 이미지 통계량 노드를 연결하고 노드 속성창의 이미지 입력 - 입력 이미지 경로가 올바른 field(원본 이미지들의 경로 정보를 가지는 field)로 지정되어 있는지 확인합니다.
    • 이미지 처리 노드와 함께 사용하는 경우, 이미지 입력 - 이미지 처리 - 이미지 통계량 순서로 연결하고 이미지 통계량 노드 속성창의 이미지 입력 - 입력 이미지 경로가 올바른 field(원본 또는 처리된 이미지들의 경로 정보를 가지고 있는 field)로 지정되어 있는지 확인합니다.
