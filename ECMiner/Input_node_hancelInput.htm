한셀 입력

한셀 입력 노드는 한셀 파일 형태로 저장된 데이터를 가져오는 노드입니다. 파일을 직접 선택하여 해당하는 데이터를 가져옵니다.

사용법, 속성

사용법

  • 한셀 파일 경로를 지정합니다.
  • 워크시트를 선택합니다.
  • 해당되는 경로의 데이터를 읽어옵니다.
  • 파일 부분 읽기 방식을 지정합니다.
  • 변수정보를 확인합니다.

속성
