평행좌표GL차트

평행좌표GL차트 노드는 데이터를 이용하여 평행좌표 차트를 그릴 수 있는 노드입니다. 평행좌표GL차트를 사용하면 여러 변수들이 어떠한 방식으로 누적되어 있는지를 확인할 수 있습니다.

사용법, 속성, 결과

사용법


  • 변수는 3개 이상 선택하여야 합니다.
  • 선택사항
    • 그룹변수 옵션 속성 변경

속성

속성그룹 속성명 설명 기타 비고
일반정보 이름 노드의 이름을 입력합니다. 선택
설명 노드에 대한 간단한 주석을 달 수 있습니다. 선택
그룹 변수 옵션 그룹표시 데이터 내에 그룹 관련 변수가 있을 때 그룹 표시 방법을 선택합니다. 그룹 사용 안함, 색상으로 구분 중 택일
그룹변수 그룹변수를 선택합니다 선택
변수선택 전체 변수 선택/해제 전체 변수를 선택하거나, 선택 해제하는 버튼으로 위의 이미지 1번 그림의 두 버튼입니다.
변수 목록 차트로 그릴 변수를 지정합니다. 위의 이미지 2번 그림에 표현된 체크박스를 이용하여, 변수를 선택할 수 있습니다. 필수
조건부 변수 선택 3번 이미지를 이용하여 변수 목록 중 제시하는 조건에 맞는 변수를 선택할 수 있습니다.

결과

  • 변수 4개 선택 시 결과


  • 각 변수 별 범위 지정
    • 각 축 마다 드래그하여 범위를 지정하면, 다음과 같이 확인할 수 있습니다.
