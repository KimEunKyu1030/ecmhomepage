기본차트

기본차트 노드는 데이터를 이용하여 일반적인 이차원 차트를 그릴 수 있는 노드입니다. 기본차트 노드를 사용하면 산점도 차트뿐만 아니라 시계열 차트도 표현이 가능합니다.

사용법, 속성, 지원 플롯

사용법

  • 차팅방식은 시계열 Y, X 대 Y(들) 중에 하나를 선택합니다.
  • '시계열 Y'는 시간척도 변수 옵션에 따라 선택한 변수의 시계열 트랜드를 나타내며, 시간척도 변수를 선택할 수 있습니다.
  • 'X 대 Y(들)'은 선택한 변수의 산점도(Scatter plot)를 나타내며, X축 변수를 1개 선택할 수 있습니다.
  • 그룹변수 옵션을 통해 그룹으로 표시할 변수와 그룹표시 방식을 선택합니다.
  • 그룹표시 방식 옵션
    • 'X 대 Y(들)' : 시리즈로 구분
    • '시계열 Y' : 시리즈로 구분, 색상으로 구분, 라벨로 표시
  • 'Y 변수 선택'에서 Y값으로 표시할 변수들을 선택합니다.
  • 선택사항
    • 변수별로 축 생성 속성 변경
    • 변수별로 Y축 분리 속성 변경
    • 그룹변수 옵션 속성 변경
    • Y축 지정 속성 변경

지원 플롯

  • X 대 Y(들) : 산점도 차트로 X, Y 축으로 지정된 두 변수 값을 사용하여 이들에 대한 이차원 차트를 그립니다.



  • 시계열 Y : 시간척도 변수나 데이터 인덱스를 X축으로, 변수의 값을 Y축으로 사용하여 이들에 대한 이차원 차트를 그립니다.
