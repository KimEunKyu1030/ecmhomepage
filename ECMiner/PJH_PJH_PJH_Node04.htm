전처리 피봇

전처리 피봇 노드는 데이터를 열(column)별, 행(row)별로 그룹화하여 분석하는 기능을 하는 노드입니다.

사용법, 속성

사용법

  • 속성창에 있는 피봇팅 정보 구성 버튼을 클릭하면 다음과 같은 속성 변경창이 뜨게 됩니다.
    • 데이터 소스가 되는 파일을 지정합니다.

    • 데이터 저장 형식을 파악하고 데이터 읽기 옵션을 선택합니다.

    • 변수형태를 지정합니다.

    • 데이터 변수와 피봇팅에 기준이 될 행, 열 변수를 지정합니다. 데이터 변수, 행, 열 변수의 조합만큼 데이터 변수가 분석되게 됩니다. 분석하고자 하는 값을 선택합니다.

    • 피봇 후 생성된 변수에 대한 목록을 확인합니다.

  • 피봇팅 정보 구성 버튼을 클릭하여 속성을 지정합니다.
  • 스트림을 실행합니다.
