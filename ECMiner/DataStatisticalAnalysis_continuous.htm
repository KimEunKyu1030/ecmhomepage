연속형 데이터 (데이터 탐색기)

데이터 탐색기에서는 데이터 통계 분석 - 연속형 데이터 기능을 사용하여, 데이터 영역내의 모든 연속형 데이터에 대한 통계량을 한번에 볼 수 있습니다.

  • 실행 방법

    • [분석] - [데이터 통계 분석] - [연속형 데이터]를 선택하면, 연속형 분포 시트가 추가됩니다.

  • 연속형 데이터 통계분석 실행 결과


  • 지원되는 통계량

    • 총합, 평균, 박스 플롯, 결측치 수, 최소값, 최대값, 범위, 분산, 표준 편차, 첨도, 왜도, 중간값, 사분위값이 지원됩니다.


  • 기타
    • NOTE
    박스 플롯의 양쪽 끝의 숫자는 outlier의 수를 나타냅니다. outlier의 수가 100개를 넘으면 '*'로 나타납니다.
