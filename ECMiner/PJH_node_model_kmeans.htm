KMEANS 모델 노드

KMEANS 모델 노드는 입력 데이터를 옵션에 따라 군집분석하여 각 데이터를 각 군집에 할당합니다.

  • 기본정보

    • 몇개의 군집으로 나뉘었는지에 대한 군집수, 수렴되기까지의 전체 시행수, 전체 관측치간 거리를 나타냅니다.

  • 군집중심정보

    • 군집분석 결과로 생성된 각 군집의 중심좌표로 각 변수들의 값들이 나타납니다.


  • 군집 내 원소 개수

    • 생성된 각 군집에 할당된 관측치의 빈도 및 백분율을 나타냅니다.


  • 군집간 거리

    • 생성된 군집의 각 군집간 거리를 나타냅니다. 이는 각 군집중심 사이의 거리를 측정한 것입니다.
