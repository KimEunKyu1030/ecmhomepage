Actor-Critic

Policy Gradient 알고리즘에서 하나의 신경망을 추가하여 두 개의 신경망을 동시에 업데이트하는 형태의 알고리즘입니다. 하나의 신경망으로는 정책(policy)을 근사하고, 다른 하나의 신경망은 Q 함수를 대체하는 구조입니다.

개요, 고려사항, 사용법, 속성, 결과

개요

  • Actor-Critic 알고리즘은 value function과 policy를 담당하는 두 개의 신경망 구조를 가집니다. Policy 구조는 action을 선택하는 역할을 수행하므로 actor라 칭하고, value function을 구하는
  • Actor는 현재 환경으로부터 t 시점의 상태(s(t))를 인식하여 정책(pi)을 통해 최적의 행동(a(t))을 산출합니다.

  • Critic은 Actor가 출력한 행동을 Q 함수를 이용하여 평가합니다.

고려사항
  • 환경모델이 필요합니다.
  • 환경모델에서 사용된 독립변수와 종속변수가 필요합니다.
  • 독립변수는 연속형/이산형의 변수 모두 사용 가능합니다.

사용법
  • 입력노드를 통해 데이터를 읽어들입니다.
  • Actor-Critic 노드를 입력노드에 연결하고 옵션들을 선택합니다.
  • 화면표시 노드를 Actor-Critic 노드에 연결합니다.
