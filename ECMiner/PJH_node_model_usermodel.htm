사용자 모델 노드

사용자 모델 노드는 파일 형태(*.GMS)로 만들어진 다른 모델 노드에 저장된 정보를 불러와, 해당 노드의 기능을 수행하는 특수한 모델 노드입니다. 주기적으로 특정 모델 파일이 자동으로 갱신되고 특정 프로젝트에서 갱신된 모델 파일을 사용하여야 하는 경우, 사용자가 해당 프로젝트 파일을 매번 수동으로 수정할 필요 없이 갱신된 모델 파일(*.GMS)의 정보를 읽어와 해당 모델의 기능을 수행하도록 할 수 있습니다.
