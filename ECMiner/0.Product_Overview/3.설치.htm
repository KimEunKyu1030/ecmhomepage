설치

ECMiner™을 설치하려면 다음과 같은 과정을 거칩니다.


  1. 제공되는 CD를 CD-ROM에 넣으면 자동 실행됩니다. 또는 ㈜이씨마이너 홈페이지인 www.ecminer.com 에서 데모 설치파일을 다운 받아 실행시킵니다. 실행화면과 동시에 다음과 같은 화면이 뜹니다.

    2. [다음(N)] 버튼을 클릭합니다.

  2. 설치할 폴더를 지정하고 [다음(N)]을 클릭합니다. 이때, 사용자 컴퓨터의 남은 디스크 용량을 확인할 수 있으며, 사용자를 정의할 수 있습니다.

    3. 최종적으로 설치 진행여부를 묻습니다. 설치를 진행하려면 [다음(N)]을 클릭합니다.

  3. 설치가 진행되며, 설치를 중도에 취소하고 싶을 경우 [취소] 버튼을 누릅니다. 설치를 계속 진행하고자 한다면 설치가 종료될 때까지 기다립니다.


  4. 모든 설치가 완료되면 설치완료 메시지가 나타나며 [닫기(C)] 버튼을 클릭하면 설치가 완료됩니다.


  5. 설치가 완료되었습니다. 바탕화면에 생성된 ECMiner아이콘을 클릭하면 ECMiner™가 실행됩니다.
