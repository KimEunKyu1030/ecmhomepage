시스템 환경
ECMiner를 설치하기 위한 시스템 사양은 다음과 같습니다.
  • 최소 사양

    CPU 인텔® 코어 2 듀오 2.4 GHz
    메모리 4 GB
    운영체제 Windows 7, Windows 8, Windows 8.1, Windows 10
    하드 디스크 240 MB + 분석하고자 하는 최대 데이터 양의 2배 이상의 여유공간


  • 권장 사양
    CPU 인텔 코어 i5 이상
    메모리 16 GB 이상
    운영체제 Windows 7, Windows 8, Windows 8.1, Windows 10
    하드 디스크 240 MB + 분석하고자 하는 최대 데이터 양의 2배 이상의 여유공간



    NOTE
    분석하고자 하는 데이터의 양에 따라 충분한 하드디스크 용량이 있어야 합니다. 예를 들어 분석하고자 하는 데이터의 크기가 500 MBytes 정도라면 최소 1 GBytes (500 MBytes x 2) 이상의 여유 공간이 하드디스크에 있어야 합니다.
