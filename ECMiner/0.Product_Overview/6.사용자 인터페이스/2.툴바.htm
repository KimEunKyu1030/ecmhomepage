툴바

메뉴와 노드창 사이에 유용한 기능을 제공하는 아이콘들의 모음인 툴바가 위치합니다. 아이콘들과 그들의 기능은 다음과 같습니다.


ICON 명칭 설명
새로 만들기
(Ctrl + N) 		프로젝트를 새로 만들 때 사용합니다.
프로젝트 열기
(Ctrl + O) 		프로젝트 파일을 열 때 사용합니다.
프로젝트 저장
(Ctrl + S) 		현재 활성된 프로젝트를 저장할 때 사용합니다. 처음 저장하는 것이라면 저장할 이름을 입력하는 대화상자가 나타납니다.
잘라내기 현재 활성된 작업창에서 선택된 노드와 노드 연결을 복사한 뒤 삭제합니다.
복사 현재 활성된 작업창에서 선택된 노드와 노드 연결을 복사합니다.
붙여넣기 현재 복사되어 있는 노드와 노드 연결을 활성된 작업창에 붙여넣습니다.
지우기 현재 활성된 작업창에서 선택된 노드와 노드 연결을 삭제합니다.
인쇄
(Ctrl + P) 		현재 활성된 작업창을 인쇄합니다.
실행
(F5) 		현재 활성된 작업창의 모든 스트림을 실행합니다.
실행 중지
(Shift + F5) 		현재 활성된 작업창이 실행 중이라면 활성되며 실행을 중지 시키고자 할 경우 사용됩니다.
부분 실행
(Ctrl + F5) 		현재 활성된 작업창에서 선택된 노드를 포함한 스트림만 부분적으로 실행합니다.
데이터 탐색기 현재 활성된 작업창에서 유효한 입력 노드가 선택되어 있을 경우 해당 노드에 대한데이터 탐색기를 나타냅니다.
노드 도움말 현재 활성된 작업창에서 선택된 노드의 도움말을 나타냅니다.
프로그램 정보 ECMiner™의 정보 대화 상자를 나타냅니다.
