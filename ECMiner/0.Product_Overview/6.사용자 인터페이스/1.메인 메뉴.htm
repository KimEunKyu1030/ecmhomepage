메인 메뉴

ECMiner™의 메인 메뉴는 다음과 같이 구성되어 있습니다.

프로젝트

  • ECMiner™의 프로젝트 파일 관리를 위한 메뉴입니다.


    • 메뉴명 설명 단축키
    새로 만들기 프로젝트를 새로 만듭니다. 이 메뉴를 선택하면 스트림 작업을 할 수 있는 작업창이 하나 새로 만들어집니다. CTRL+N
    열기 작업하고자 하는 프로젝트를 선택하여 엽니다. 메뉴를 선택하면 프로젝트 파일을 선택할 수 있는 대화상자가 나타납니다. CTRL+O
    닫기 현재 선택되어 있는 프로젝트(작업창)을 닫습니다. CTRL+F4
    저장 현재 선택되어 있는 프로젝트(작업창)의 내용을 저장합니다. 처음 저장하는 것이라면 이름을 입력할 수 있는 대화상자가 나타나고 저장했던 프로젝트라면 기존에 저장했던 파일을 덮어 씁니다. CTRL+S
    다른 이름으로 저장 현재 선택되어 있는 프로젝트(작업창)의 내용을 다른 이름으로 저장하고자 할 경우 선택합니다. 저장할 이름을 입력하는 대화상자가 나타납니다. -
    인쇄 현재 선택되어 있는 작업창의 작업 내용을 인쇄합니다. CTRL+P
    인쇄 미리 보기 작업창의 내용을 인쇄하기 전 어떤 형태로 인쇄되는 지 확인할 수 있습니다. -
    인쇄 설정 인쇄할 때 필요한 옵션을 지정합니다. -
    끝내기 ECMiner™을 종료합니다. 종료할 경우 저장되지 않은 프로젝트에 대하여 저장할 것인지 여부를 묻습니다. ATL+F4
  • 기 작업했던 프로젝트 파일은 4개까지 최근 사용한 파일로 프로젝트 메뉴에 나타납니다. 작업을 연속적으로 할 때 편리한 기능입니다.

편집

  • ECMiner™ 프로젝트 편집 시 필요한 편집 기능을 제공합니다.


    • 메뉴명 설명 단축키
    잘라내기 현재 활성화 되어 있는 작업창 내에서 선택된 노드, 노드 연결을 잘라냅니다. (복사하고 지움) CTRL+X
    복사 현재 활성화 되어 있는 작업창 내에서 선택된 노드, 노드 연결을 붙여넣기 위하여 복사합니다. 선택된 것이 없을 경우 활성화되지 않습니다. CTRL+C
    붙여넣기 현재 복사되어 있는 노드, 노드 연결을 활성화 되어 있는 작업창에 붙여넣습니다. 복사된 것이 없다면 활성화 되지 않습니다. CTRL+V
    지우기 현재 활성화 되어 있는 작업창 내에서 선택된 노드, 노드 연결을 삭제합니다. 선택된 것이 없을 경우 활성화되지 않습니다. DELETE

보기

  • ECMiner™ 사용자 인터페이스 상의 여러 컨트롤창을 보이게/안보이게 할 수 있습니다.


    • 메뉴명 설명 단축키
    도구 모음 ECMiner™ 인터페이스 부분에서 언급했듯이 데이터 마이닝 작업을 위한 여러 칸트롤창이 제공됩니다. 이 컨트롤 창 중 필요 없는 것을 안보이게 하거나 다시 보이게 하는 기능을 수행합니다.

    위와 같은 서브 메뉴가 있으며 각 컨트롤창 메뉴를 선택하여 보이게/안보이게 할 수 있습니다.     		-
    상태 표시줄 상태 표시줄을 보이게/안보이게 합니다. -
    스트림 맵 현재 구성된 프로젝트의 스트림을 보여주는 맵을 표시합니다. -
    데이터 탐색기 현재 선택된 작업창에서 유효한 입력 노드가 선택되었을 경우 활성화 됩니다. 이 메뉴를 선택하면 데이터 탐색기가 나타나고 통계적인 데이터 분석을 수행할 수 있습니다. -

스트림

  • ECMiner™ 프로젝트의 기본이 되는 스트림을 편집하고 실행하기 위한 기능을 제공합니다.


    • 메뉴명 설명 단축키
    노드 추가 노드를 추가하고자 할 경우 사용합니다. 노드창에서 노드를 추가할 수도 있으며 같은 기능입니다. 메뉴를 선택하면 노드 분류별로 서브 메뉴가 나타나고 서브 메뉴 중 하나를 선택하면 해당 분류에 속한 노드가 메뉴화 되어 나타납니다. -
    노드 생성후 연결 노드 추가와 비슷한 기능으로 다른 점은 노드를 추가한 후 현재 작업창에 선택된 노드에 자동으로 연결한다는 것입니다. 만약 현재 선택된 노드가 연결될 수 없는 것이라면(출력 노드 등) 서브 메뉴가 나타나지 않습니다. -
    연결 현재 선택된 노드를 시작점으로 하여 다른 노드에 연결할 때 사용합니다. 이 메뉴보다 마우스 가운데 버튼을 활용하는 것이 더 편리합니다. -
    연결 끊기 현재 작업창에 선택되어 있는 연결을 끊습니다. -
    실행 현재 작업창에 구성된 모든 스트림을 실행합니다. F5
    실행 중지 현재 작업창(프로젝트)가 실행 중일 경우 실행을 중지하고 싶을 때 사용합니다. SHIFT+F5
    부분 실행 현재 작업창 내의 모든 스트림을 실행하는 것이 아니라 선택된 노드를 포함한 스트림만 실행합니다. 노드가 선택되어 있지 않은 경우 활성화 되지 않습니다. CTRL+F5

확장기능

  • 실험계획법(DOE)와 확률분포 분석을 수행할 수 있는 확장된 기능입니다.


    • 메뉴명 설명 단축키
    DOE 실험계획법 기능을 제공합니다. -
    확률분포 다양한 확률 분포에 관한 모수, 확률분포, 차트 등의 기능을 제공합니다. -

도구

  • ECMiner™ 환경 설정을 할 수 있는 메뉴입니다.


    • 메뉴명 설명 단축키
    옵션 ECMiner™ 실행 및 UI를 위한 옵션을 지정/편집합니다. -

창

  • 현재 열러 있는 작업창을 전환하거나 정렬할 수 있는 기능을 제공합니다.


    • 메뉴명 설명 단축키
    계단식 배열 현재 열려 있는 작업창들을 계단식으로 배열합니다. -
    바둑판식 배열 현재 열려 있는 작업창들을 바둑판식으로 배열합니다. 프로젝트 간 스트림 구성등을 비교해 보고자 할 때 유용합니다. -
    프로젝트1 현재 열려 있는 프로젝트 목록이 나타납니다. 메뉴명은 프로젝트 이름을 표시하므로 변경될 수 있습니다. 최대 9개까지 나타나며 나타난 것 이외의 것을 활성화하고자 할 경우는 Windows 메뉴를 이용합니다. -
    Windows 현재 열린 프로젝트 목록을 대화상자 형태로 나타냅니다. 목록 중 원하는 프로젝트를 선택하여 활성화 하거나 저장하는 등의 기능을 수행할 수 있습니다. -

정보

  • ECMiner™에 대한 정보 대화상자를 표시하기 위한 메뉴입니다.


    • 메뉴명 설명 단축키
    ECMiner 정보 ECMiner™의 정보 대화상자를 나타냅니다. 간단한 Credit, 버전 및 빌드 번호 등을 확인할 수 있습니다. -
