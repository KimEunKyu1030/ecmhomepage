수식 편집기
수식 편집기는 파생변수, 선택, 채우기 등 다양한 노드에서 사용되며, 새로운 값을 계산하거나 조건을 위한 수식을 편집할 때 사용합니다. 마우스 조작만으로 수식을 편집할 수도 있으며 익숙해진 경우 수식 편집창에 직접 입력함으로써 보다 빠르게 수식을 편집할 수 있습니다.

특징

  • 마우스 조작만으로 쉽게 ECMiner™에서 사용할 수식을 편집할 수 있습니다.
  • 수식의 각 요소에 따라 색상을 달리 표시함으로써 판독성을 높였습니다.
  • 다양한 수식 편집이 가능합니다.
  • 저장 / 불러오기 기능을 이용하여 기 작성된 수식을 재 사용할 수 있습니다.
  • 줄 단위 주석문자("//")를 이용하여 줄 단위의 주석을 추가할 수 있습니다.

사용자 인터페이스

번호 설명 비고
(1) 툴바. 실행취소/다시실행, 찾기/바꾸기, 책갈피 설정 등 수식 편집을 위한 기능을 툴바 형태로 선택할 수 있습니다.
(2) 수식 편집창. 수식을 편집할 수 있는 곳입니다. 임의로 입력할 수도 있으며 (4) ~ (8)을 이용하여 편집할 수도 있습니다. (4) ~ (8)을 이용하면 마우스 조작만으로 쉽게 ECMiner™에서 사용할 수식을 편집할 수 있습니다.
(3) 현재 입력되어 있는 수식을 저장하거나 예전에 저장해 두었던 수식을 불러 와 다시 사용할 수 있습니다. 지우기 버튼을 누르면 현재 입력되어 있는 수식이 모두 지워집니다. 확장자: *.eaf
(4) ECMiner™에서 제공하는 함수의 범주를 선택할 수 있습니다. ECMiner™에서는 변환함수, 수학/삼각, 텍스트, 날짜/시간, 변수통계, 정보 관련 함수들을 제공합니다. 이를 변경할 때마다 지정된 범주에 해당하는 함수들이 (4)에 목록됩니다. 함수 목록 참조
(5) 지정된 함수범주에 해당하는 함수들이 나타납니다. 사용하고자 하는 함수를 선택하여 더블클릭하거나 추가 버튼을 누르면 수식편집창에 입력됩니다. (8)에 각 함수의 사용법 및 간단한 설명이 나타납니다.
(6) IF ~ THEN ~ ELSE ~ ENDIF 구문을 입력합니다.
(7) 현재 파생변수 노드에서 사용할 수 있는 기존 변수 목록입니다. 입력하고자 하는 변수를 선택한 뒤 더블클릭하여 편집창에 추가할 수 있습니다. 추가하면 {DATE} 같이 "{}"이 붙어 나타나며 이는 변수임을 의미하게 됩니다.
(8) 마우스 조작만으로 수식을 편집할 수 있도록 하기 위해 많이 사용되는 숫자와 기호를 버튼으로 구성한 부분입니다. 버튼을 누르면 해당하는 숫자 혹은 기호가 편집창에 입력됩니다.
(9) 현재 선택되어 있는 함수에 대한 간단한 설명 및 사용법이 나타납니다.
(10) 현재 입력되어 있는 구문이 올바르게 되었는지 검사합니다. 구문이 틀렸을 경우 오류 메시지를 반환하며 이 오류 메시지를 보고 잘 못된 부분을 수정할 수 있습니다.
(11) 현재 입력되어 있는 구문을 입력하거나 취소합니다. 확인 버튼을 누르면 입력되며, 입력하기 전에 구문검사를 수행합니다. 만약 잘못된 구문이 입력되었을 경우 프로그램이 제대로 동작하지 않을 수도 있습니다.

유용한 기능

기타 수식 편집에 유용한 기능들


  • 수식 편집 자동 완성

    • 수식 편집 작성에 사용되는 함수, 키워드, 매크로, 데이터 필드명등의 초기 문자열 (또는 접두사)을 입력 한 후 다양한 종류의 텍스트를 자동으로 완성하므로 긴 단어를 모두 입력 할 필요가 없으며 잠재적으로 잘못 입력하지 않아도 됩니다.

    • 함수 자동 완성

      • 수식 편집 작성에 사용되는 함수나 키워드의 첫문자나 문자열 입력시 해당 문자나 문자열로 시작되는 함수 또는 키워드 목록을 표시합니다.

    • 매크로 자동 완성

      • 수식 편집 작성에 사용되는 매크로의 첫문자 '@' 입력시 매크로 목록을 표시합니다.

    • 데이터 필드 자동 완성

      • 데이터 필드 구분기호 첫문자 '{' 입력시 사용 가능한 데이터 필드 목록을 표시합니다.

  • 열 편집

    • Alt + 마우스 드래그 또는 Alt + Shift + 화살표 키를 사용하여 열 모드로 전환할 수 있습니다.

  • 줄 단위 주석

    • 줄의 제일 처음에 줄단위 주석 문자 ("//") 를 추가하여 해당 줄을 주석 처리할 수 있습니다.

색상

  • 문자열 -> "123"
  • 날짜형 -> #2004-01-01#
  • 숫자형 -> 123
  • 함수, IF 구문 -> ABS()
  • 변수 -> {DATE}
  • 연산자, 매크로 -> +, -, *, /, @ROWNUM
  • 오류 -> ABS(
  • 주석 -> // 주석입니다...

관련항목

  • 함수 목록
  • 정의된 매크로
  • 값 입력
