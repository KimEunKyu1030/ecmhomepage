Word Cloud

Word Cloud 노드는 데이터를 이용하여 단어의 중요도나 사용 빈도를 직관적으로 파악할 수 있도록 시각화하여 표현하는 노드입니다.

사용법, 속성, 결과

사용법


  • 단어 리스트 변수 속성 창에서 Word Cloud에 사용할 문자형 변수를 선택합니다.
  • 단어 개수 리스트 변수 속성 창에서 단어 리스트 변수에서 선택한 변수의 개수를 가진 변수를 선택합니다.
  • 차트 형태 속성 창에서 Spiral 또는 Typewirter를 선택합니다.

속성

속성그룹 속성명 설명 기타 비고
일반정보 이름 노드의 이름을 입력합니다. 선택
설명 노드에 대한 간단한 주석을 달 수 있습니다. 선택
차트 기본 옵션 단어 리스트 변수 Word Cloud에서 사용할 변수를 선택합니다. 문자형
단어 개수 리스트 변수 단어 리스트 변수에서 사용되는 변수의 개수를 가진 변수를 선택합니다. 필수
차트 형태 Spiral 또는 Typewriter 중 하나를 선택합니다. Spiral, Typewriter

결과

  • Spiral

    • Spiral 기능은 단어들의 개수를 상대적으로 배교하여 개수가 많은 단어들 순으로 차트의 중앙에 근접하게 위치하게 하며, 단어의 크기는 점점 작하지는 형태로 표시한다.


  • Typewriter
    • 기능은 단어의 개수가 가장 많은 단어부터 나열하며, 크기 또한 점점 작아지는 형태로 표시한다.
