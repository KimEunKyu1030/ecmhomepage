t-SNE

t-SNE(t-distributed Stochastic Neighbor Embedding)은 비선형적 차원 축소를 통하여 저차원상에서 데이터의 분류 및 분포를 시각화하는 다변량 자료분석기법입니다. 기존의 선형적 차원 축소 방법론들(PCA, MDS 등)에 비하여 데이터 분류의 시각화에 있어 우수한 성능을 가집니다.

개요, 고려사항, 사용법, 속성, 결과

개요
  • t-SNE는 PCA와 마찬가지로 자료가 펼쳐진 방향 또는 경향을 발견함으로써 자료가 내포하고 있는 정보를 발견할 수 있습니다. 고차원의 많은 변수들로 표현되는 원본 data points 간의 유사도를 보존하면서 차원을 축소합니다. 대부분의 경우 축소하고자 하는 저차원은 시각화에 수월한 2차원 또는 3차원입니다.
  • 기존의 선형적 방식을 이용한 차원 축소 방법론은 원본 data points를 저차원으로 mapping할 때, 원본 data에서 군집화 되어 있는 data points가 뭉쳐지는 문제가 발생합니다. t-SNE는 비선형적 차원 축소 방법론으로, 선형적 mapping 방식으로는 불가능한 영역의 data 분류 및 시각화 성능을 보입니다.
  • 관측치 N 개, 독립변수 D 개인 input matrix X를 행렬 대칭화, gradient descent 등을 이용하여 d(≤ D) 차원 상의 data points 행렬로 mapping합니다. 이렇게 mapping된 points를 map points 라고 칭합니다.
  • Algorithm이 동작할 때마다 input data 중 기준점을 임의로 선정하는 등의 random process 가 포함되어 있습니다. 따라서 input data 및 설정이 동일하더라도 매 algorithm 동작 시 마다 출력되는 결과(d 차원의 map points)에 다소 차이가 발생하게 됩니다.

고려사항
  • 독립변수만 입력되어야 하며 연속형이어야 합니다.
  • t-SNE 노드 자체에 표준화 전처리과정을 포함하고 있습니다.

사용법
  • 입력노드를 통해 데이터를 읽어들입니다.
  • t-SNE 노드를 입력노드에 연결하고 옵션들을 선택합니다.
  • t-SNE 노드 설정값 중 Initial Perplexity 는 input data 개수(input matrix 행 개수)의 1/3 미만이어야 합니다.
  • t-SNE 노드는 모델 파일을 생성하지 않습니다.

속성
속성그룹 속성명 설명 기타 비고
일반정보 이름 노드의 이름을 입력합니다. 선택
설명 노드에 대한 간단한 주석을 달 수 있습니다. 선택
선택사항 수행횟수 총 학습 반복 횟수를 설정합니다. 필수 0 이상의 충분히 큰 자연수. 기본값은 1000 입니다.
Initial Perplexity 초기 Perplexity 값을 설정합니다. 필수 5 ~ 50 사이의 값을 가지도록 합니다. 단 input data 개수(input matrix 행 개수)의 1/3 미만이어야 합니다.
학습률 상수 학습률 상수 값을 설정합니다. 필수 기본값은 100 이며, 경우에 따라 조정하도록 합니다.
Threshold 연산의 정밀도를 설정합니다. 필수 Threshold 값이 0 에 가까울수록 정확도는 높아지나, 연산 시간이 길어집니다. 0 이상의 실수 값으로 설정합니다. 기본값은 0.5 입니다.
Initial Momentum 초기 momentum 값을 설정합니다. 필수 학습 횟수가 250에 도달할 때 까지는 Initial Momentum 값을 학습에 사용합니다. 기본값은 0.5 입니다.
Final Momentum 최종 momentum 값을 설정합니다. 필수 학습 횟수가 250을 초과한 이후부터는 Final Momentum 값을 학습에 사용합니다. 기본값은 0.8 입니다.
축소 차원 수 축소하고자 하는 저차원의 차원 수를 설정합니다. 필수 시각화에 용이한 차원은 2차원 또는 3차원이므로, 대개 2 또는 3으로 설정합니다. 기본값은 2 입니다.

결과

    (1) 분석결과정보

    화면표시 노드에서 원본 data points 정보와 축소 차원 상의 map points의 좌표 정보를 확인할 수 있습니다.

    (2) 분석결과 시각화

    차트 노드, 화면표시 노드 등과 연동하면 2차원 또는 3차원으로 시각화할 수 있습니다.
