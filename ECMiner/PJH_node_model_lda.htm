LDA 모델 노드

LDA 모델 노드는 트레이닝 데이터를 러닝하여 새로운 데이터의 분류에 사용될 수 있습니다.

  • 클래스별 독립변수평균

    • 분류된 각 클래스의 변수평균값을 나타냅니다.


  • 클래스의 분산-공분산행렬

    • 그룹별 분산-공분산이 같기 때문에 한 개의 분산-공분산 행렬이 출력됩니다.


  • 사전확률

    • 점유도 옵션에서 지정한 클래스별 확률값을 보여줍니다.


  • 그룹별 선형판별함수

    • 각 class에 대한 선형함수의 상수와 계수를 나타냅니다.


  • Confusion matrix for number & percentage

    • 원 클래스 인덱스와 분류분석을 통해 얻어진 클래스 인덱스에 대한 빈도와 퍼센트 교차표를 보여줍니다. 또한, 잘못 분류된 관측치의 수와 비율을 나타냅니다.
