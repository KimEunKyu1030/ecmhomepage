매트릭스차트

매트릭스차트 노드는 데이터를 이용하여, 매트릭스차트를 그릴 수 있는 노드입니다. 매트릭스차트를 사용하면, 여러 변수들의 상관 관계를 하나의 그래프로 표현할 수 있습니다.

사용법, 속성, 지원 플롯

사용법

  • 차팅방식은 단순행렬도와 X, Y 지정행렬도
    • 를 선택합니다.
  • 단순행렬도는 선택한 변수를 X와 Y축 모두에서 사용합니다.
  • X, Y 지정 행렬도는 변수 선택 #1에서 선택한 변수를 X축으로 사용하고, 변수 선택 #2에서 선택한 변수를 Y축으로 사용합니다.
  • 그룹변수 옵션을 통해 그룹으로 표시할 변수와 그룹표시 방식(색상으로 구분)을 선택합니다.
  • 선택사항
    • 그룹변수 옵션 변경
    • X축, Y축 지정 속성 변경

변수선택 #1 전체 변수 선택/해제 전체 변수를 선택하거나, 선택 해제하는 버튼으로 위의 이미지 1번 그림의 두 버튼입니다.
변수 목록 매트릭스차트로 그릴 변수를 지정합니다. 위의 이미지 2번 그림에 표현된 체크박스를 이용하여, 변수를 선택할 수 있습니다. 필수
조건부 변수 선택 3번 이미지를 이용하여 변수 목록 중 제시하는 조건에 맞는 변수를 선택할 수 있습니다.
변수선택 #2 전체 변수 선택/해제 차팅 방식이 X, Y 지정 행렬도일 경우 활성화됩니다. 전체 변수를 선택하거나, 선택 해제하는 버튼으로 위의 이미지 4번 그림의 두 버튼입니다.
변수 목록 차팅 방식이 X, Y 지정 행렬도일 경우 활성화됩니다. 매트릭스차트로 그릴 변수를 지정합니다. 위의 이미지 6번 그림에 표현된 체크박스를 이용하여, 변수를 선택할 수 있습니다. 필수
조건부 변수 선택 차팅 방식이 X, Y 지정 행렬도일 경우 활성화됩니다. 5번 이미지를 이용하여 변수 목록 중 제시하는 조건에 맞는 변수를 선택할 수 있습니다.

지원 플롯

  • 단순행렬도 : 지정된 변수를 X, Y축으로 사용하여 산점도 차트를 매트릭스 형태로 그립니다.



  • X, Y 지정행렬도 : 변수선택 #1에서 지정된 변수를 X축으로 사용하고, 변수선택 #2에서 지정된 변수를 Y축으로 사용하여 산점도 차트를 매트릭스 형태로 그립니다.
