매트릭스GL차트

매트릭스GL차트 노드는 데이터를 이용하여, 매트릭스차트를 그릴 수 있는 노드입니다. 매트릭스GL차트를 사용하면, 여러 변수들의 상관 관계를 하나의 그래프로 표현할 수 있습니다.

사용법, 속성

사용법


  • 히스토그램 Bin 개수 옵션은 입력된 개수의 수 만큼 히스토그램의 구간 개수가 설정됩니다.
  • 변수는 3개 이상 선택하여야 합니다.
  • 선택사항
    • 그룹변수 옵션 속성 변경

속성

속성그룹 속성명 설명 기타 비고
일반정보 이름 노드의 이름을 입력합니다. 선택
설명 노드에 대한 간단한 주석을 달 수 있습니다. 선택
차트옵션 히스토그램 Bin 개수 히스토그램의 구간 개수를 설정합니다. 자연수
그룹 변수 옵션 그룹표시 데이터 내에 그룹 관련 변수가 있을 때 그룹 표시 방법을 선택합니다. 그룹 사용 안함, 시리즈로 구분 중 택일

변수 목록 차트로 그릴 변수를 지정합니다. 위의 이미지 2번 그림에 표현된 체크박스를 이용하여, 변수를 선택할 수 있습니다. 필수
