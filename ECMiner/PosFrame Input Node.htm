포스프레임 입력

포스프레임 입력 노드는 포스프레임 입력 노드는 포스코 분석 Workbench 사용자 편의성을 개선하기 위해서 개발되었습니다. 포스코 측에서 API를 통해 상용툴 데이터 추출을 지원할 때, 해당 노드를 활용하여 포스프레임(하둡) 데이터를 직접 조회하거나 분석워크벤치용 마트데이터를 선택하여 지원하는 데이터를 csv파일 형태로 받아오는 역할을 수행합니다.

사용법, 속성

사용법

  • 사용 유형(가동계, 개발계) 및 게이트웨이 종류(포항, 광양)를 추출 데이터 종류에 맞게 선택합니다.
  • API 자원 분류를 입력합니다. (ex. p-m00, k-m00 등)
  • API 사용자 직번을 입력합니다.
  • API 활용 유형을 선택합니다. (마트 데이터 추출, 포스프레임(하둡) 데이터 직접 추출)
  • 파라미터 설정 버튼을 눌러서 데이터 추출에 필요한 세부적인 속성값들을 설정합니다.
  • 변수정보를 확인합니다.

  • 좌측 상단(1)에 나열된 선택가능한 4개의 체크박스를 활용하여 condition parameters의 사용 여부를 설정합니다. 체크한 파라미터의 활성화된 에디트 박스에 원하는 값을 입력한 후에 마트 리스트 조회(2) 버튼을 클릭하여 해당 조건으로 마트 리스트를 조회합니다.
  • 마트 리스트를 조회하면 해당 정보가 좌측 하단(3) 리스트로 출력됩니다. 출력된 마트 리스트 중에 원하는 마트 데이터를 클릭하면 포커싱이 잡히면서 해당 데이터의 컬럼 리스트를 조회합니다.
  • 우측 상단(4)에 선택한 마트 데이터의 컬럼 리스트가 출력됩니다. 이 때, 우측 하단의 ‘확인’ 버튼을 누르면 파라미터 설정이 완료되고, 해당 마트 데이터를 csv파일 형식으로 가져옵니다.

  • 가장 먼저 좌측 상단(1)에서 불러올 데이터 유형을 선택합니다. 선택한 데이터 유형에 따라서 적합한 파라미터를 채워넣습니다.
  • 우측 상단(2)에서는 condition parameters 중에서 필수 파라미터 외에 사용자가 직접 지정한 파라미터 이름, sql 명령어, 값 등을 이용하여 추가할 수 있습니다.
  • IN, >= Value =< (SQL 명령어)에서 에디트 박스에 두 개 이상의 값을 입력할 때에는 “A, B, C”와 같이 쉼표를 이용해서 구분하여 표기하도록 합니다.
