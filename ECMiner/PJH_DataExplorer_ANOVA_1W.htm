일원 배치법: Stack

일원배치 분산 분석은 하나의 인자에 대한 분산 분석 방법입니다. 인자와 관측치를 선택하시면 분산 분석을 하실 수 있습니다.

분산 분석 다이얼로그 열기

  • 메뉴의 [분석]을 선택한 후, [분산분석]을 다시 선택하면 3가지 분산 분석 종류가 나열됩니다. 그중에서 일원배치법:Stack을 선택하시면 됩니다.

Stack 데이터

  • Stack 데이터 : 아래의 Figure 1과 같이 모든 데이터를 하나의 필드에 쌓아서 정리한 데이터 형태를 말합니다.

일원 배치 : Stack 데이터

  • 인자 : 그룹을 구분짓는 이산형인 변수를 선택합니다.
  • 관측치 : 그룹별로 일원배치 분산분석을 실시할 변수를 선택합니다. 
    Figure 1 일원 배치법 : Stacked 데이터

인자 및 관측치 선택방법

  • 마우스 드래그를 이용하여 선택할 수 있습니다. 먼저 왼쪽의 필드 리스트서 필드를 선택한 후, 오른쪽의 인자 리스트 또는 관측치 리스트로 마우스로 드래그앤드롭을 하면 필드가 왼쪽에서 오른쪽 리스트로 이동됩니다.

사후 분석 및 신뢰도

  • 일원배치의 사후 분석 방법으로는 4가지 사후 분석 방법이 제공되고 있습니다. 필요에 따라 원하는 사후 분석방법을 체크 하시면 됩니다. 신뢰도는 0.05와 0.01 두 가지가 있습니다. 필요에 따라 원하는 신뢰 구간을 선택 하시면 됩니다.
    • LSD : The Least Significant Difference Method
    • Tukey : Tukey's Test (by Tukey(1953))
    • Scheffe : Scheffe's Method for Comparing All Contrasts
    • Duncan : Duncan's Multiple Range Test
