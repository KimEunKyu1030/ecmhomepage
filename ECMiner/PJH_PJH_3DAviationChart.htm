3차원항적차트

3차원항적차트 노드는 데이터를 이용하여 3차원항적차트를 그릴 수 있는 노드입니다.

사용법, 속성

사용법

  • 두 개 이상의 시리즈(비행 경로)를 그릴 경우, 필요한 시리즈(비행 경로) 개수만큼 시리즈 추가 버튼을 눌러 추가합니다.
  • 비행기 목록 속성창에서 X축, Y축, Z축, PK로 사용할 변수를 선택합니다.
  • 선택사항
    • 슬라이드 표현 속성 변경
    • PK 임계값 속성 변경

속성

속성그룹 속성명 설명 기타 비고
일반정보 이름 노드의 이름을 입력합니다. 선택
설명 노드에 대한 간단한 주석을 달 수 있습니다. 선택
차트옵션 시리즈 추가 여러 시리즈를 한 차트에 그리기 위해 사용됩니다. 버튼을 누르면 시리즈 선택 리스트가 추가됩니다. 버튼
시리즈 삭제 마지막에 추가된 시리즈가 삭제 됩니다. 버튼
시리즈 편집 선택되지 않은 시리즈를 삭제합니다. 버튼
슬라이드 표현 차트의 슬라이드 정보에 표시될 값을 지정합니다.
PK 임계값 입력된 임계값 이상의 PK 값을 가진 좌표를 차트 상의 포인터로 표시합니다. 기본값으로 95가 입력되어 있습니다.
시리즈 목록 X축 X축으로 사용할 변수를 지정합니다.
Y축 Y축으로 사용할 변수를 지정합니다.
Z축 Z축으로 사용할 변수를 지정합니다.
PK 차트 상의 PK 포인트와 마크 정보에 Kill Rate 값으로 표현할 변수를 지정합니다.
