ElmanNet 모델 노드
ElmanNet 모델 노드는 ElmanNet 분석의 결과로 생성됩니다. Training 데이터 셋으로 학습하여 새로운 데이터의 예측에 사용될 수 있습니다.

  • 옵션 정보

    • ElmanNet를 이용해서 모델링 시 어떤 옵션을 사용했는지를 보여줍니다. 이것을 통해 옵션이 바뀌었을 때 결과가 어떻게 바뀌었는지 판단할 수 있습니다.


  • ANOVA 테이블

    • 회귀분석의 경우에만 해당됩니다. 독립변수가 종속변수 y를 설명하는데 있어서 통계적으로 유의한지 검정하기 위해 F검정을 합니다. p-값이 유의수준보다 커서 유의하지 않다면 회귀계수에 대한 해석은 의미가 없습니다.

  • 표준에러정보
    1. R-square(결정계수)

      2. 도출된 회귀식이 측정값들을 대표할만한가를 확인하는 것으로, 즉, 추정된 회귀선이 관측치들을 얼마나 잘 설명하는지를 검정하는 값입니다. 1에 가까울수록 회귀식이 데이터를 잘 설명하고 있음을 의미합니다.도출된 회귀식이 측정값들을 대표할만한가를 확인하는 것으로, 즉, 추정된 회귀선이 관측치들을 얼마나 잘 설명하는지를 검정하는 값입니다. 1에 가까울수록 회귀식이 데이터를 잘 설명하고 있음을 의미합니다.

    2. Root-MSE (Root-Mean Square Error)
    3. MAE (Mean Absolute Error)
    4. MPE (Mean Absolute Percentage Error)
    2~4 번 모두 종속변수의 원래값과 추정값의 오차를 나타낸 것으로 값이 작을수록 좋은 모형이라할 수 있습니다.
