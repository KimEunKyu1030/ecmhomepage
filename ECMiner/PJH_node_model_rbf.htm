RBF 모델 노드

RBF 모델 노드는 트레이닝 데이터를 러닝하여 새로운 데이터의 예측이나 분류에 사용될 수 있습니다.

  • 옵션정보

    • RBF 이용해서 모델링시 어떤 옵션을 사용했는지를 보여줍니다. 이것을 통해 옵션이 바뀌었을때 결과가 어떻게 바뀌었는지 판단할 수 있습니다.

  • 중심

    • Radial Basis Function의 중심값입니다.


  • 오분류정보

    • 분류의 경우에만 해당됩니다. 원클래스와 예측클래스를 보여줘 틀린관측치 수와 백분율을 알 수 있습니다.


  • ANOVA 테이블

    • 회귀분석의 경우에만 해당됩니다. 독립변수가 종속변수 y를 설명하는데 있어서 통계적으로 유의한지 검정하기 위해 F검정을 합니다. P-값이 유의수준보다 커서 유의하지 않다면 회귀계수에 대한 해석은 의미가 없습니다.


  • 표준에러정보

    • 회귀분석의 경우에만 해당됩니다.


    1. R-square(결정계수)

      2. 도출된 회귀식이 측정값들을 대표할만한가를 확인하는 것으로, 즉, 추정된 회귀선이 관측치들을 얼마나 잘 설명하는지를 검정하는 값입니다. 1에 가까울수록 회귀식이 데이터를 잘 설명하고 있음을 의미합니다.

    2. Adjust R2

      3. 투입되는 독립변수들이 많을수록 종속변수에 대한 설명력인 R-square는 자연히 증가합니다. 그러므로 실제 적합도를 판정하는 데에는 자유도를 고려하여 조정된 결정계수 R-square(adj)를 많이 활용합니다. 1에 가까울수록 회귀식이 데이터를 잘 설명하고 있음을 의미합니다.

    3. Root-MSE (Root-Mean Square Error)
    4. MAE (Mean Absolute Error)
    5. MPE (Mean Absolute Percentage Error)

    ③~ ⑤번 모두 종속변수의 원래값과 추정값의 오차를 나타낸 것으로 값이 작을수록 좋은 모형이라할 수 있습니다.
