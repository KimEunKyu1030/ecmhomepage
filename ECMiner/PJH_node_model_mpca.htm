MPCA 모델 노드

MPCA 모델 노드는 모델로서의 기능은 없습니다. 즉, 모델링 자체가 분석이 되는 경우입니다.

  • MPCA 모델 노드에 입력되는 data는 modeling에 사용된 분석용 data set과 같은 형식으로 전처리 및 unfolding이 수행된 data set이어야 합니다.
  • MPCA 수행정보

    • 전체변수를 몇 개의 주성분으로 축약했는지 표시해주고 T^2의 Limit를 알 수 있습니다.

  • 로드값

    • 각 변수를 이용해 주성분을 생성하는 선형결합을 표시하고 있는 값으로 선형결합시 계수값을 의미합니다. 즉, 주성분 1은 다음과 같이 생성됩니다. 주성분1 = (-0.2383*y3)+(-0.08679*x1)+(0.03601*x2)+…


  • 분산설명력

    • 각 주성분이 전체데이터에 대한 정보를 어느 정도 가지고 있는지를 표시하고 있습니다.


  • Loading plot

    • Loading plot은 처음 두개의 주성분(주성분1과주성분2)의 loading값에 대한 정보를 제공합니다.


  • Scree Plot

    • Scree plot은 각 주성분에 대한 Eigenvalue값을 표시한 것입니다. Scree plot은 주성분의 수를 결정하는 하나의 방법으로 Eigenvalue값의 변화가 거의 없는 곳에서 주성분의 수를 결정할 수 있습니다.


  • Time Step Chart

    • MPCA 모델 노드에만 지원되는 기능으로 MPCA 모델 노드에 연결된 입력 data에 대한 Time Step Chart를 보여줍니다. MPCA 모델링 노드 결과보기의 Time Step Chart와 같은 사용자 인터페이스로 구성됩니다.
    Trend Chart 상의 붉은 선은 MPCA 모델링 노드에 입력된 batch들에 대한 해당 변수의 평균값의 time steps에 따른 trend입니다.
    MPCA 모델 보고서에서 Time Step Chart를 확인하기 위해서는 적합한 입력 data에 연결된 후 1회 실행되어야 합니다.
    위 과정을 수행하지 않고 모델 보고서 실행시 Time Step Chart가 출력되지 않습니다.
