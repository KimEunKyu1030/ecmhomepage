Python 연동

Python 연동 노드는 local 경로의 Python 실행 환경과 연동하여 입력 data에 대해 Python script를 실행하고 결과를 출력하는 기능을 수행합니다.

개요, 고려사항, 사용법, 속성, 속성창

개요
  • Python 연동 노드는 local에 설치된 Python 실행 환경(Conda의 Python 가상 환경 포함)과 연동하는 기능을 수행합니다.
  • 스크립트 편집기를 이용하여 이미 작성된 Python script(*.py)를 불러오거나 직접 입력할 수 있습니다.
  • 입력 data를 pandas DataFrame 형식으로 변환하고 Python script를 수행하여 결과를 출력합니다.

고려사항
  • Python 환경 항목에서 Python 구동 환경을 선택합니다.
  • 스크립트 항목의 편집 button을 click하면 스크립트 편집기 대화 상자가 출력됩니다.
  • Numpy, Pandas library를 기본적으로 불러옵니다. Numpy는 np, Pandas는 pd로 불러옵니다. 따라서 노드 동작을 위해서는 사용하고자 하는 환경에 기본적으로 이들 두 개의 library가 설치되어 있어야 합니다.
  • 입력 data는 ecmData 라는 이름의 pd.DataFrame 형식으로 저장됩니다. 따라서 입력 data에 대하여 작업을 수행하기 위해서는 pandas DataFrame 형식의 data에 적용할 수 있는 함수를 이용하여야 합니다.
  • 마찬가지로 노드 동작 후에는 ecmData pd.DataFrame 내에 존재하는 data를 출력하게 됩니다.
  • 노드 실행 직전에는 필수적으로 출력 변수 목록을 갱신해주어야 합니다.

사용법
  • 입력노드를 통해 data를 읽어들입니다.
  • 입력 data에 대하여, 필요한 경우 여러 전처리 노드를 추가적으로 연결하여 Python 연동 노드를 동작하고자 하는 data로 정제합니다.
  • Python 연동 노드를 연결하고, 노드 속성창의 Python 환경 항목의 Python 환경 ComboBox에서 연동하고자 하는 Python 환경을 선택합니다. 선택한 환경에 대한 정보, 환경 위치, Python 실행 파일 경로, Pythonw 실행 파일 경로가 아래에 표시됩니다.
  • 노드 속성창의 스크립트 항목의 편집 button을 click하면 스크립트 편집기 대화 상자가 출력됩니다. 스크립트 편집기에서는 Python script를 새로 입력하거나 불러오기 기능을 이용하여 이미 작성된 Python script file(*.py)을 불러올 수 있습니다.
  • 날짜형 data를 다루기 위해 datetime module을 import하여 사용하도록 설정되어 있습니다.
  • 기본적으로 Numpy library는 np, Pandas library는 pd로 불러오도록 설정되어 있습니다. 따라서 numpy와 pandas에 대해서는 library를 받아오기 위해 별도로 import 명령할 필요가 없습니다. 이외의 다른 library들이 필요한 경우는 스크립트 편집기에서 별도로 import 명령문을 통하여 받아오도록 합니다.
  • ecmData라는 이름의 pandas DataFrame 객체를 이용하여 data를 처리합니다. 기본적으로 ecmData는 노드에 입력되는 data를 가지고 있습니다.
  • 따라서, Python script 상에서 입력 data에 대하여 작업을 수행하기 위해서는 ecmData에 대하여 pandas DataFrame 형식의 data를 다루는 데에 사용되는 함수를 이용하여야 합니다.
  • 마찬가지로 노드 동작 후에는 ecmData 내에 존재하는 data를 출력하게 됩니다. Python script 상에서 새로 생성되는 변수 또는 결과값을 출력 data로 확인하고자 한다면 ecmData pd.DataFrame에 추가해주어야 합니다.
  • 노드 실행 직전에는 반드시 변수 목록 갱신 기능을 이용하여 출력 변수 목록을 갱신해주어야 합니다. 갱신 후 변수 목록에 표시되는 변수들은 입력 data에 대한 Python script 동작 후 최종적으로 ecmData pd.DataFrame 내에 존재하는 변수들입니다.

선택사항 스크립트 Python script를 작성합니다. 선택 스크립트 편집기를 이용하여 Python script를 입력할 수 있습니다. 기존에 작성된 script를 불러올 수도 있습니다.
Python 환경 Python script를 동작시킬 Python 환경을 선택합니다. 선택 Conda에서 생성된 가상 환경들을 포함하여 이용 가능한 Python 환경들 중 사용하고자 하는 환경을 ComboBox 형식으로 선택할 수 있습니다.

필수사항 변수 목록 갱신 변수정보에 표시되는 변수 목록을 갱신합니다. 필수 스크립트 편집기에서 Python script를 수정한 후 변수 목록 갱신 button을 click하여 수정된 script에 따른 출력 변수 목록을 갱신합니다. Python 연동 노드 동작 전에는(특히 스크립트 편집기의 Python script를 수정하였을 때) 필수적으로 변수 목록 갱신 button을 click하여 변수 목록을 갱신하여야 합니다.

속성창

    (1) 노드 속성창

    노드 속성창에서 노드 설정 정보를 변경할 수 있습니다.

    (2) 스크립트 편집기

    노드 속성창의 편집 button을 click하면 스크립트 편집기 대화창이 출력됩니다. 기존의 Python script(*.py)를 불러오거나 직접 입력하여 작성할 수 있습니다.

    (3) Python 환경 선택

    노드 속성창의 Python 환경 ComboBox를 이용하여 Python 환경을 선택할 수 있습니다. Conda 등에서 만들어진 가상 환경들을 포함하여 선택할 수 있습니다.

    (4) 변수 목록 갱신

    노드 속성창의 변수 목록 갱신 button을 click하면 변수 목록을 갱신할 수 있습니다. 입력 변수들에 대하여 Python script를 동작시킨 후 최종적으로 출력되는 변수들의 목록 및 데이터형이 갱신됩니다.
