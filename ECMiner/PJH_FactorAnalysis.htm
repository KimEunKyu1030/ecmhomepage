Factor Analysis

Factor Analysis는 여러 변수들에서 공통된 요인을 추출하는 방법론입니다.

개요, 고려사항, 사용법, 속성, 결과

개요

  • Factor Analysis는 여러 변수 들에서 공통된 요인을 추출하는 방법론입니다. Factor Analysis는 관측되지 않은 공통된 요인(Common Factor)들을 찾고 이 공통 요인들이 각 관측된 변수에 어떠한 영향을 미치는지를 알고자 하는 것이 목적입니다.
  • 예를 들어 여러 학생의 영어, 국어, 사회, 수학, 과학, 기술 성적이 있다고 합니다. 상식적으로 생각해 볼 때 수학을 잘하는 학생들은 과학, 기술을 잘하는 경우가 많고 국어를 잘하는 학생들은 영어, 사회를 잘하는 경우가 많습니다. 이는 아마도 수학, 과학, 기술 성적은 인간의 계량적인 능력에 의해 영향을 많이 받고, 국어, 영어, 사회는 인간의 언어적인 능력에 영향을 많이 받기 때문이라 추측해 볼 수 있습니다.


  • 기술된 내용을 수식으로 표현하면 위와 같습니다. 이 때 x는 각 과목의 성적을 나타내고, μ는 각 과목 성적의 모평균을 말합니다. 그리고 F1을 Common Factor1(공통 요인1), F2를 Common Factor2(공통 요인2) 라고 하고 ε를 공통 요인들로 설명되지 않는 부분이라고 합니다. 국어, 영어 사회, 성적에 대해서 λ1의 값이 크고, λ2은 값은 작고, 또한 수학, 과학, 기술 성적에 대해 λ2의 값이 크고, λ1의 값이 작다면 F2을 계량적 능력을 나타내는 하나의 지표가 되는 요인으로, F1를 언어적 능력을 나타내는 하나의 지표가 되는 요인으로 해석할 수 있습니다. 이 λ값의 집합은 Factor Loading 값으로 표현 됩니다.


  • 위의 식에서 m은 총 변수수가 해당되고 공통 요인 Fi 는 지정된 공통 요인 개수 만큼 계산 가능합니다. 각 데이터에 대하여 공통 요인을 구하기 위해서는 Load 값을 활용하여 Fi 값을 산출 가능하고, 위의 βi 값은 Load 값을 통하여 확인 가능합니다.

고려사항

  • 독립변수만 입력되어야 하며 연속형이어야 합니다.
  • 종속변수 추가 시 분석에 활용이 되지 않습니다.
  • Factor Analysis노드 자체에 표준화 전처리 과정을 포함하고 있습니다.

  • Factor Analysis노드를 입력노드에 연결하고 옵션들을 선택합니다.

결과

    (1) 분석을 실행한 Diagram 상에서 Factor Analysis 노드를 클릭해서 노드 속성창의 결과보기버튼을 누르면 다음 결과들이 출력됩니다.


  • Control Chart

    • Control Chart를 통해 회전 및 비회전시의 분포를 확인 할 수 있습니다.


  • Loading Plot

    • Loading plot은 선택된 두 개의 원 데이터 혹은 요인의 loading값에 대한 정보를 제공합니다.


  • Scree Plot

    • Scree plot은 각 요인에 대한 Eigenvalue값을 표시한 것입니다. Scree plot은 요인의 수를 결정하는 하나의 방법으로 Eigenvalue값의 변화가 급격히 이루어지는 에서 요인의 를 결정할 수 있습니다.


  • Contribution

    • 1. control chart에서 편차가 커 보이는 두 그룹을 분석 목표로 정한 후, 우선 첫번째 그룹을 위 그림에서 보이는 것과 같이 ①처럼 드래그 한 뒤, ②버튼을 클릭해서 앞서 드래그한 테이터를 추출합니다. 두 번째 그룹에도 같은 과정을 반복합니다.

    2. 데이터 선택이 끝났으면 Contribution Plot 부분으로 가서 공헌도를 파악하고 싶은 요인과, 그룹 데이터를 선택합니다. 선택 후 그리기 버튼을 누르면 아래와 같은 Contribution Plot이 출력됩니다.
